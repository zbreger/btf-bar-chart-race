Notes about the above checkboxes

Use the checkboxes above only when taking screenshots. You should refrain from using the associated query parameters when embedding because delay in JavaScript execution will result in an initial flash of content and potential content layout shift.

If you wish to hide certain text, either delete the appropriate tags from the HTML template or remove the property from your configuration file.

If you wish to use homepage styles, either set the body's class name to homepage or set the entry's output property in the configuration file to homepage.
